CPCC ITS offers Citrix as our remote access offering for both students and employees. There are three methods of connecting to CPCC resources from the Citrix platform. All of them are accessible from http://remote.cpcc.edu.

  1. Virtual Desktop – is a hosted computer environment which provides access to CPCC resources from anywhere on all popular platforms.
  2. Virtual Application – is a hosted application service which provides direct access to CPCC licensed applications from anywhere on all popular platforms with the need to log into a computer.

We recommend using Mozilla Firefox or Google Chrome when using CPCC Cloud.
